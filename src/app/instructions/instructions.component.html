Welcome to Entities Comparison
In the left menu you can select "Comparison" and write the entities from Dbpedia that you want to compare.
First, you can write the URL of the main entity and then as many other entities as you want.
There are some default values ​comparing Aristotle and Socrates.
Note that the Dbpedia URL must be valid.
There is a button to delete all current fields and there are buttons to add or delete other entities.
When you've entered all the entities you want to compare, you can click the "Start" button.

There are five modes you can compare your entities.
The first mode is "Wikipedia" and is the original wikipedia site for the selected entities.
The second mode is "Triples" where you can see all the information about the selected entities.
On the left is the main entity and on the right are all the other entities. You can change them with the "Previous" and "Next" button.
The third mode is "Specific predicates" and there the users can select from the selection box a specific predicate that they want to see.
Again, on the left is the main entity and on the right are all the other entities.
The forth mode is "Images" and there you can see all the images that exists in Dbpedia for those entities.
Finally, the fifth mode is "Commonalities" and the user can see how much the entities have in common.
